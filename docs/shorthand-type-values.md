# Shorthand Type Values

Shorthand values for [built-in types](/src/Type) are determined by lower-casing the class name and removing the "Type"
suffix.
For example:

| Class          | Shorthand Value |
|----------------|-----------------|
| TextType       | text            |
| NumberType     | number          |
| EmailType      | email           |
| CollectionType | collection      |
